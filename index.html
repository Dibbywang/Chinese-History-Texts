About the Project

This website is dedicated to making Chinese historical texts more available for the English-speaking audience. Provided are some Chinese historical texts translated into English. They are far from perfect. If you see any section that requires polishing, please contact the author by clicking here to copy the email. We welcome more collaborators on GitHub to make the project better.
